    세이지 스킬

    델류즈

    계열 : 지면 보조
    습득 조건 : 프로스트 웨폰 2LV
    내용 : 해당 지역을 델류즈지역으로 만든다.

    레벨 데미지 증가 MaxHP 고정 캐스팅 변동 캐스팅 소모 SP 지속시간 효과범위
    1 + 10% + 5% 0.5 초 3 초 48 60 초 7 * 7 셀
    2 + 14% + 9% 46 120 초
    3 + 17% + 12% 44 180 초
    4 + 19% + 14% 42 240 초
    5 + 20% + 15% 40 300 초

    ※ 사용 시, 촉매로 블루 젬스톤 1개를 소모한다.

    ※ 지면 마법 사용 시, 장판스킬 활성 상태에서는 지면 마법을 사용할 때에 촉매가 소모되지 않는다.

    　단, 장판스킬 활성 상태의 지속시간은 다시 지면 마법을 사용하여도 갱신되지 않는다.

    ※ 델류즈 지역 위에서는 모든 수속성 공격의 공격력이 증가하며, 몸 속성이 수속성인 대상의 MaxHP가 증가한다.

    ※ 델류즈 지역에서 [아쿠아 베네딕타]와 [워터 볼]을 사용할 수 있다.

    ※ 델류즈 위에서 워터볼을 사용하면, 워터볼로 끌어온 술자 주변의 델류즈가 사라진다.

    ※ 세이지의 다른 지면 마법과 중첩 사용이 불가능하다.

    드래고놀로지

    계열 : 패시브
    습득 조건 : 어드밴스드 북 9LV
    내용 : 용족에 대한 연구를 통해, 용족에 대한 내성 과 용족에게 주는 물리/마법 데미지 를 높인다.

    　　　스킬레벨이 증가함에 따라 INT가 최대 3까지 증가한다.

    2 + 8% + 8% + 4%
    3 + 12% + 12% + 6% + 2
    4 + 16% + 16% + 8%
    5 + 20 + 20 + 10% + 3

    습득 조건 : 스펠 브레이커 3LV
    내용 : 옐로우 젬스톤 1개를 소모하여, 일정 확률로 대상에게 부여되어있는 모든 마법 효과를 해제한다.

    레벨 성공 확률 고정캐스팅 변동캐스팅 소모 SP

    2 70%
    3 80%
    4 90%

    ※ 5LV 미만의 스킬을 사용할 경우, 대상의 MDEF에 따라 성공 확률이 감소한다.

    ※ PVP, 공성 지역에서만 플레이어에게 사용할 수 있다.

    종류 재료
    화속성 컨버터 전갈의 꼬리 3개 + 빈 스크롤 1개
